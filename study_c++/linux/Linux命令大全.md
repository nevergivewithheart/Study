# Linux命令大全

#### 1、ssh远程连接

- 登录到远程主机上

  ```shell
  #登录：端口为22，用户名user，远程连接ip：192.168.0.123
  ssh -p22 user@192.168.0.123
  ```

- 登出从远程主机

  ``` shell
  ##登出：用户名user，远程连接ip：192.168.0.123
  ssh -X user@192.168.0.123
  ```

- 在终端中断开远程连接，不关闭终端

  ```shell
  #logout 只能退出当前登录用户的角色
  ctrl+D
  ```
